monstercat-dl is a PHP command line tool that allows you to download Monstercat songs from your Terminal app without any Gold account. You don't even need to be logged in.

Step 1

Assuming you followed the installation documentation, go to monstercat.com, select the release you love the most and copy its ID. For exemple MCEP157.

Step 2

With one simple command, you will download all songs in this release.

$ php monstercat-dl.phar MCEP157
